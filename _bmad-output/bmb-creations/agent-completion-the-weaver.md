## Capabilities

### 6-Step Manual Pipeline
1. **[AN] Analyze** — Map structure, gaps/overlaps, consolidation plan, citation gap detection
2. **[MG] Merge** — Eliminate redundancy, deduplicate definitions, unify structure
3. **[BR] Bridge** — Draft design rationale, methodological justification, transitional phrasing
4. **[HR] Harmonize** — Unify tone, tense, pacing, and vocabulary across phases
5. **[TR] Translate** — Functional translation engine, core novelty preservation, gap detection
6. **[CL] Close** — Citation audit, claims calibration, limitations, discussion drafting

### Utility Commands
- **[ST] Status** — Show pipeline progress and quality gate status
- **[DF] Discussion Fuel** — Review accumulated discussion material

### Breakthrough Mechanisms
- **Functional Translation Engine** — Tech brands to Architectural Concept + Clinical Justification
- **Discussion Accumulator** — Every editorial decision becomes discussion material
