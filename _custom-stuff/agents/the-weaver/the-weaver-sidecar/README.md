# the-weaver-sidecar

This folder stores persistent memory for **The Weaver** agent.

## Purpose
Maintains pipeline state for thesis consolidation across sessions.

## Files
- `memories.md` — User profile, session history, preferences
- `instructions.md` — Operating protocols, annotation patterns, pipeline rules
- `consolidation-plan.md` — Proposed unified ToC with structural rationale
- `discussion-fuel.md` — Running buffer of editorial decisions as discussion material
- `pipeline-status.md` — Quality gate results, step progress, core novelty flags, student decisions

## Runtime Access
After BMAD installation, this folder will be accessible at:
`{project-root}/_bmad/_memory/the-weaver-sidecar/`
