# dsr-writer-sidecar Sidecar

This folder stores persistent memory for the **Dr. Peffers** agent.

## Purpose
Agent requires persistent memory setup and proactive context loading for DSR thesis drafting

## Files
- memories.md: User profile, session history, patterns
- instructions.md: Protocols, boundaries, startup behavior

## Runtime Access
After BMAD installation, this folder will be accessible at:
`{project-root}/_bmad/_memory/dsr-writer-sidecar/{filename}.md`
